# ActionUI Elements

This index lists all ActionUI elements. Each element has:
- **Schema** (`.md`) — full JSON specification with comments documenting every property, defaults, and observable state
- **Template** (`.json`) — clean JSON ready to copy-paste into a view composition

All views inherit baseline properties from [View](Schemas/View.md) ([template](Elements/View.json)) — padding, frame, opacity, foregroundStyle, background, font, cornerRadius, shadow, border, accessibility, actionID, etc.
